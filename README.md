#vscode::shortcut keys
shift + delete : 한줄삭제
ctrl + D : 커서위치 음절 선택
shift + alt + ↑↓ : 선택영역 복사
alt + ↑↓ : 선택한 줄 이동
ctrl + alt + ↑↓ 컬럼모드
shift + alt + a : <!--  -->생성
ctrl + ` : terminal cursor
alt + number : tab cursor move
ctrl + number : section cursor move/create
ctrl + alt + ←,→ : section move
ctrl + n : new file(undefined)
ctrl + f4 : close tab
